The Boldt Style Guide

January 20, 2006 University Web Office

Fonts

Default body fonts are: Arial and Helvetica. Specialized font for page title, interior menu items and subheadings on the Homepage is Bodega Sans Old Style.

Colors

The default (standard paragraphs, headings) text color is a dark gray: #363636.

The pallette for The Boldt uses several variations of green and a deep blue as an accent, their Hex/Web values are:

  • Deep Blue - #413F4C (used in main page headings - h2)
  • Green - #6F6F6D (used in other page headings - h3, h4, h5, etc
  • Khaki Green - #8E8C83 (used for links)
  • Tan - #E2D8D0 (#E2D8D0) - Used as the Background in the Header and the homepage right column
  • Deep Green - #4C5352

Image Sizes

The border around the images is rendered in CSS, and does not need to be included in the image file.

  • Homepage Banner Image: 403 x 121 pixels
  • Interior Banner Image: 549 x 68 pixels
  • Homepage Feature Story Thumbnail: 180 x 141 - this image size is flexible as long as the width conforms to the column widths outlined below.
  • Homepage Featured Alumni: 103 x 104 pixels
  • Homepage News Thumbnails: 80 x 65 pixels

Column Widths

  • Left Navigation: 140px
  • Homepage Right Column: 140px
  • Homepage Center Column: 350px
  • Interior Page Content Column: 520px

Homepage Banners

The path to the Homepage Banners folder is: images/photos/headers/homepage/*.jpg
